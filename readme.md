
## Sala Multimèdia Intel·ligent

Aplicació per simplificar l’ús de les sales multimèdia en entorns professionals.

## Descripció

Disseny i desenvolupament d’un sistema interactiu per a la gestió eficient de sales de reunions híbrides en entorns hospitalaris i professionals.

## Objectiu

Simplificar l’ús dels dispositius tecnològics, especialment per a usuaris no tècnics, reduint incidències i temps d’inici.

## Funcionalitats

- Inici de videotrucades (Microsoft Teams)
- Accés a documentació (Office Online)
	
Test de dispositius:
- Càmera
- Micròfon
- Sistema d’audio
- Pantalla
- Test general de la sala
- Sistema de notificació d’incidències

## Tecnologies utilitzades
	
### FRONTEND
- HTML5
- CSS
- JavaScript
	
### APIs DEL NAVEGADOR
	
- MediaDevices API (Accés a Càmera i Micròfon)
- WebAudio API (Anàlisi del nivell d’àudio)
	
### INTEGRACIONS
	
- Microsoft Teams
- Microsoft Office Online

## Execució del projecte

- Descarregar el projecte o clonar el repositori
- Obrir el fitxer `index.html` amb un navegador web (recomanat Google Chrome o Microsoft Edge)
- Permetre l’accés a dispositius (càmera i micròfon) si es sol·licita

## Requeriments

- Navegador web (Chrome, Edge o Firefox)
- Accés a càmera i micròfon per les proves
- Connexió a internet (per accedir a Teams i Office)

## Limitacions

- No disposa de backend ni persistència de dades
- Validació realitzada en entorn controlat
